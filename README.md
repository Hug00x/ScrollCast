# ScrollCast — uma descrição para utilizadores

ScrollCast é uma aplicação pensada para quem gosta de ler e anotar documentos de forma natural — como se estivesse a usar um caderno físico, mas com o poder de multimédia e organização digital.

Com o ScrollCast consegues ler PDFs ou cadernos próprios e adicionar anotações visuais, notas de texto, gravações de áudio e imagens diretamente sobre cada página.

## Para quem é o ScrollCast?

- Para estudantes que querem sublinhar e comentar apontamentos.
- Para professores que querem preparar feedback visual em documentos.
- Para pessoas que tomam notas visuais, esboços e gravações por página.

## O que podes fazer com ela

- Desenhar à mão livre sobre qualquer página (suporte a caneta e a dedo).
- Criar notas de texto que ficam associadas a pontos da página.
- Gravar notas de voz e colocá‑las como pinos na página.
- Inserir imagens (por exemplo recortes, fotos) e posicioná‑las sobre a página.
- Guardar automaticamente o que fazes por página — quando voltares, a app reabre na última página onde estavas.
- Escolher cores de uma paleta ou introduzir um código HEX (caso queiras uma cor exacta).
- Ver e reutilizar as cores mais recentes usadas nesse documento.
- Adicionar ou remover páginas em cadernos (com confirmação para evitar perdas acidentais).

## Como começar

1. Abre a app e escolhe um PDF ou um caderno.
2. Navega para a página que queres anotar.
3. Usa a barra de ferramentas para escolher cor, espessura do traço, ou alternar para o modo borracha.
4. Toca e arrasta para desenhar; usa os botões de desfazer/refazer conforme precisares.
5. Para adicionar uma nota de texto, gravação ou imagem, usa os botões correspondentes na interface.


## Privacidade e armazenamento

- As anotações e os ficheiros (ex.: imagens, áudio) são guardados localmente no dispositivo.
- A app não envia automaticamente os teus documentos nem as tuas notas para servidores externos.
